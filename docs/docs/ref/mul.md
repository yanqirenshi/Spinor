# *
**Kind:** Function  
**Signature:** `(Int, Int) -> Int`
### Syntax:
```lisp
(* a b)
```
### Arguments and Values:
- `a` -- 第一オペランド
- `b` -- 第二オペランド
- 戻り値: `a * b`
### Description:
2つの整数を乗算します。
### Examples:
```lisp
(* 3 4)   ; => 12
(* 7 0)   ; => 0
```
