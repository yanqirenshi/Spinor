# Numbers

Spinor における数値型と算術演算について解説します。

## 数値型

Spinor は以下の2つの数値型をサポートしています。

### 整数 (Int)

任意精度の整数値です。内部的には Haskell の `Integer` 型として表現されます。

```lisp
42          ; 正の整数
-17         ; 負の整数
0           ; ゼロ
1000000000  ; 大きな整数
```

### 浮動小数点数 (Float)

倍精度浮動小数点数です。内部的には Haskell の `Double` 型として表現されます。

```lisp
3.14        ; 正の小数
-2.5        ; 負の小数
0.0         ; ゼロ
1.0e10      ; 指数表記
```

## 算術演算

### 基本演算

```lisp
(+ 1 2)       ; => 3   加算
(- 5 3)       ; => 2   減算
(* 4 5)       ; => 20  乗算
(/ 10 3)      ; => 3   整数除算
(mod 10 3)    ; => 1   剰余
```

### 比較演算

```lisp
(= 1 1)       ; => t   等値
(< 1 2)       ; => t   小なり
(> 2 1)       ; => t   大なり
(<= 1 1)      ; => t   以下
(>= 2 2)      ; => t   以上
```

## 型変換

整数と浮動小数点数が混在する演算では、自動的に浮動小数点数に変換されます。

```lisp
(+ 1 2.5)     ; => 3.5 (Float)
(* 2 3.0)     ; => 6.0 (Float)
```

## 型推論

数値リテラルの型は文脈から推論されます。

```lisp
spinor> 42
:: Int
42

spinor> 3.14
:: Float
3.14

spinor> (+ 1 2)
:: Int
3
```
