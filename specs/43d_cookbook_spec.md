# 43d: Spinor Cookbook (逆引きレシピ集) 仕様書

## 1. 概要
Spinor の機能を組み合わせた実践的なコード例（レシピ）を提供し、ユーザーが言語のポテンシャルを最大限に引き出せるようにする。本ドキュメントは、単なる機能紹介ではなく「やりたいこと」から逆引きできる形式を目指す。

## 2. アーキテクチャ
- **ファイルパス:** `manual/public/docs/cookbook.md`
- **レンダリング:** 既存のマニュアルサイトの `react-markdown` 基盤を利用。

## 3. コンテンツ構成 (主要レシピ)

### 3.1 安全なファイル処理 (Resource Protection)
`read-file`, `write-file` と `unwind-protect` を組み合わせた、例外に強いリソース管理パターン。
- 例: ファイルを読み込み、処理中にエラーが発生しても必ず終了ログを出力する。

### 3.2 JSON データの抽出と変換 (Web API Mockup)
Alist としてパースされた JSON データから特定のキーを抽出し、Spinor のデータ型へ変換するパターン。
- 例: `(json-parse ...)` の結果から `assoc` 等を使って特定フィールドを取り出す。

### 3.3 並行ワーカーパターン (Concurrency with MVar)
`spawn` と `MVar` を使用した、スレッド間通信と状態同期のパターン。
- 例: メインスレッドから計算タスクを投げ、ワーカースレッドが結果を `MVar` に書き込んで同期する。

### 3.4 科学技術計算と GPU 演算 (Matrix & OpenCL)
行列の基本演算と、OpenCL カーネルを用いた高速計算のパターン。
- 例: 行列積の計算と、`cl-enqueue` を用いた並列加算。
