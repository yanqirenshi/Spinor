# 仕様: Spinor プロモーション用Webサイト

## 1. 概要

Spinor プロジェクトの公式ランディングページを作成し、プロジェクトの認知度向上とコミュニティ形成の第一歩とする。この Web サイトは、プロジェクトのコンセプト、主な特徴、そして始め方を簡潔に伝えることを目的とする。

## 2. 目的

*   **コンセプトの伝達:** "静的型付け Lisp" という Spinor の核心的なアイデンティティを訪問者に明確に伝える。
*   **情報集約:** プロジェクトに関する主要な情報（GitHub リポジトリ、ドキュメント、ロードマップ）への入り口として機能させる。
*   **第一印象の向上:** クリーンでモダンなデザインにより、プロジェクトの信頼性と専門性を高める。

## 3. ターゲットオーディエンス

*   静的型付け言語に興味を持つ Lisp プログラマー
*   関数型プログラミング（特に Haskell）に興味を持つ開発者
*   新しいプログラミング言語の探求者

### 4.3. 特徴セクション (Features Section)

Spinor の主要な特徴を3〜4点、アイコンと共に簡潔に紹介する。

*   **静的型付け (Static Typing):**
    *   説明: コンパイル時の型チェックにより、実行時エラーを未然に防ぎ、コードの堅牢性を高める。Hindley-Milner 型推論をサポート。
*   **Haskell の意味論 (Haskell Semantics):**
    *   説明: 純粋関数、遅延評価、代数的データ型（ADT）、パターンマッチングなど、Haskell の強力な機能を Lisp の構文で利用できる。
*   **ブートストラップ (Self-Hosting):**
    *   説明: コア部分は Haskell で実装し、標準ライブラリは Spinor 自身で記述。言語の自己拡張性を実証。
*   **軽量な並行処理 (Lightweight Concurrency):**
    *   説明: Haskell の軽量スレッド（Green Threads）と MVar を活用し、シンプルで安全な並行プログラミングをサポート。

### 4.4. コード例セクション (Code Example Section)

Spinor の文法と表現力を示す、簡潔で分かりやすいコード例を提示する。シンタックスハイライトを適用すること。

**例:**
```lisp
;; Define a type-safe length function for lists
(defun length ((xs (List a))) : Int
  (match xs
    ('() 0)
    ((cons _ rest) (+ 1 (length rest)))))

;; Calculate the length of a list of integers
(length '(1 2 3 4 5))
;; => 5
```
